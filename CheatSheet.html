CSS Selector Cheat Sheet

Selector Description Example
Color Changes the forground colour eg. text colour Color: Tomato;
Background-Color Changes the background colour Background-color: DodgerBlue;
Border Specifies the border do go around a piece of content border: 2px Dotted Black
Margin Sets the distance between the border and another object margin: auto;
Padding Sets the distance between a piece of content and the border around it padding: auto;
Text Align Aligns the text within the box of content text-align: right;
Font-Family Sets the style of font to be used on a piece of text Font-Family: Helvetica;
Font-Weight Sets how thick the piece of text is font-weight: 1000;
